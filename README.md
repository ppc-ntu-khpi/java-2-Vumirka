# Практична робота "Створення і використання статичних методів"
## Завдання №22: Перетворити задане число з двійкової системи на [код Грея](https://uk.wikipedia.org/wiki/%D0%9A%D0%BE%D0%B4_%D0%93%D1%80%D0%B5%D1%8F)
### Опис завдання
Метою роботи є реалізація методу `Calculate` у класі `Exercise` для перетворення заданого десяткового числа у **код Грея**.

Код Грея - це система числення, де кожен наступний код відрізняється від попереднього лише одним бітом. Застосовується у цифровій техніці та обчислювальних алгоритмах.

---

### Деталі реалізації
У класі `Exercise` створено статичний метод, який виконує наступні дії:

1. Приймає вхідне число `R`.
2. Виконує побітову операцію XOR між числом та його зсувом на один біт вправо:
```java
int gray = R ^ (R >> 1);
```
